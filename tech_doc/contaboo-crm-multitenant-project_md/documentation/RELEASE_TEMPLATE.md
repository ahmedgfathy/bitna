## 🚀 **Migration Guide**

### **New Installation**
```bash
git clone https://github.com/ahmedgfathy/contaboo.git
cd contaboo
npm install
cp .env.example .env
npm run db:push
npm run dev
```

### **Existing Users**
```bash
git pull origin main
npm install
npm run db:migrate
npm run build
```
